# ConvertEasy

Free online unit converter tools built with Next.js 14 and Tailwind CSS.

## Features

- Cooking measurement converter (cups to grams)
- Height converter (cm to feet/inches) with 51 pre-generated pages
- Clothing size converter (US/UK/EU/Asia)
- Temperature converter (Celsius/Fahrenheit/Kelvin)
- Mobile-responsive design
- SEO optimized with sitemap generation
- AdSense-ready ad slots

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```

## Deploy

This project is configured for deployment on [Vercel](https://vercel.com).
